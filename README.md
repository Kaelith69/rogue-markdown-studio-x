# Rogue Markdown Studio X

A premium, feature-rich, single-page offline-first Markdown Editor and Live Preview Studio. Engineered for high performance, custom styling preferences, LaTeX mathematical typesetting, and Mermaid diagram workflows.

## Key Features

- **Dual-Engine Rendering:** 
  - **Online Mode (Enhanced):** Powered by `marked.js`, `DOMPurify` (XSS Sanitized), `KaTeX` (mathematical expressions), and `Mermaid.js` (flowcharts & diagram workflows).
  - **Offline Mode (Standard):** Powered by a built-in block-level Markdown parsing engine, ensuring full offline reliability without external CDN dependency.
- **Interactive Checklists:** Checkboxes in the live preview sync clicking actions directly back to the editor Markdown source text.
- **Scroll Synchronization:** Bi-directional scroll synchronization links editor and preview scroll ratios smoothly with lock prevention.
- **Custom Themes:** Choose between multiple curated color palettes:
  - *Midnight Slate* (Default)
  - *Rogue Brutalist*
  - *Cyberpunk Neon*
  - *Earthy Forest*
  - *Solarized Light*
- **Sleek Interface & Settings:** Includes layout controls (Split, Editor-only, Preview-only), typography configurations, and a real-time character/word/reading time metrics counter.
- **Print Optimization:** Embedded `@media print` styling overrides formats document downloads into clean, publication-ready PDF reports.

## Getting Started

1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Start writing Markdown!
